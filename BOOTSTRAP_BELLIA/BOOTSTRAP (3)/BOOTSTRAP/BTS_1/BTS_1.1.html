A cosa serve bootstrap?

Per uno sviluppo web più rapido e semplice

Cosa devi importare per creare una pagina web con bootstrap?

Devi importare tutti i link necessari per creare quella pagina web bootstrap

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Content Delivery Network, serve per collegare tra loro tutti i link. Il download di bootstrap sul sito comporta tempi di caricamento più rapidi

A cosa serve questo attributo del tag meta? content="width=device-width, initial-scale=1

Per garantire il corretto rendering e il tocco dello zoom

Che differenza c’è tra Container e Container Fluid?

La Container classe fornisce un contenitore a larghezza fissa reattiva. La Container Fluid classe fornisce un contenitore a larghezza intera , che copre l'intera larghezza del viewport
